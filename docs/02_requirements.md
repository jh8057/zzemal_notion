# Daily Assistant — 요구사항

## 기능 요구사항

### Must Have

| ID   | 기능                    | 설명                                         |
| ---- | ----------------------- | -------------------------------------------- |
| F-01 | 아침 브리핑 자동 발송   | 매일 아침 정해진 시간에 이메일 발송          |
| F-02 | Notion 관심사 기반 뉴스 | Notion에 작성한 관심 키워드/주제로 뉴스 수집 |
| F-03 | 구글 캘린더 일정 포함   | 오늘의 캘린더 일정 브리핑에 포함             |
| F-04 | 스몰스탭 목록 포함      | Notion의 스몰스탭 목록 가져와서 포함         |
| F-05 | Gemini API 인사이트     | 일정/스몰스탭 내용 기반으로 관련 정보 첨부   |
| F-06 | LeetCode 문제 + 답 포함 | 매일 대표 LeetCode 문제 1개와 풀이 포함      |
| F-07 | 저녁 브리핑 자동 발송   | 매일 저녁 정해진 시간에 이메일 발송          |
| F-08 | 미완료 스몰스탭 알림    | 오늘 완료하지 않은 스몰스탭 목록 포함        |
| F-09 | 목표 리마인더           | Notion의 목표 목록 포함                      |
| F-10 | MCP 서버 질문 응답      | "오늘 뭐해야돼?" 등의 질문에 Claude가 답변   |

| F-11 | 목표 기반 스몰스탭 자동 생성 | 저녁 브리핑 시 목표+미완료 스텝 기반으로 내일 스몰스탭을 Gemini가 제안 → Notion 자동 작성 |
| F-12 | 목표 추천              | 저녁 브리핑에 기존 목표 기반 새 목표 아이디어 제안 (Notion 작성은 사용자가 직접 결정) |

### Nice to Have

| ID   | 기능       | 설명                     |
| ---- | ---------- | ------------------------ |
| F-21 | 주간 회고  | 주 1회 한 주 회고 브리핑 |

## 비기능 요구사항

- **비용**: 월 0원 목표 (GitHub Actions 무료 티어)
- **신뢰성**: 아침/저녁 브리핑 누락 없이 발송
- **보안**: API 키 GitHub Secrets로 관리, 코드에 하드코딩 금지
